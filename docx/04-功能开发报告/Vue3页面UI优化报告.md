## 设计风格统一

### 配色方案
```css
--color-primary: #d95468;    /* 玫瑰红 */
--color-secondary: #ed7654;  /* 橙红色 */
--color-accent: #FFBAAC;     /* 浅粉色 */
--color-bg-card: #ffffff;    /* 卡片背景 */
```

### 背景渐变
```css
background: linear-gradient(to bottom right, #fef7f3, #fdf4f5, #fce7f3);
```

### 通用样式元素

1. **卡片样式**
   - 圆角：1rem 或 1.5rem
   - 阴影：0 4px 6px -1px var(--color-shadow)
   - 悬停：transform translateY(-4px) + 增强阴影

2. **按钮样式**
   - 圆角：9999px（完全圆角）
   - 渐变背景或纯色背景
   - 悬停抬起效果

3. **动画效果**
   - 淡入动画：fadeIn
   - 悬停动画：scale/translateY
   - 过渡时间：0.3s ease

4. **响应式设计**
   - 移动优先
   - 合理的断点设置
   - 网格自适应布局

## 移动端适配

### 响应式断点
- 小屏幕：< 640px
- 中等屏幕：640px - 1024px
- 大屏幕：>= 1024px

### 适配要点
1. 网格列数自适应
2. 字体大小缩放
3. 内边距和间距调整
4. 图标和按钮大小优化

## 性能优化

### 动画性能
- 使用 `transform` 而非 `position`
- 使用 CSS 动画而非 JS
- 合理的动画延迟

### 图片优化
- 使用渐变色代替封面图
- 减少 HTTP 请求
- 懒加载准备
