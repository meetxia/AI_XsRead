## 📋 项目概述

### 1.1 项目背景

**文字之境** 是一个女性向小说阅读平台,目前由您作为唯一作者运营。为了更好地管理平台内容、了解用户行为、监控运营数据,需要开发一套功能完善的后台管理系统。

### 1.2 系统定位

- **使用对象**: 您(作者/平台管理员)
- **核心目标**: 
  - 📊 全面的数据统计与分析
  - 📚 高效的内容创作与管理
  - 👥 用户行为监控与管理
  - ⚙️ 灵活的平台配置
  - 📈 实时的运营决策支持

### 1.3 技术特点

- **前后端分离架构**
- **响应式设计** - 支持PC、平板、手机访问
- **实时数据** - WebSocket推送关键指标
- **安全防护** - 管理员权限验证、操作日志

### 生产部署

#### 1. 前端构建
```bash
cd admin-frontend
npm run build
# 生成 dist/ 目录
```

#### 2. Nginx配置
```nginx
server {
    listen 80;
    server_name admin.yourdomain.com;

    # 前端静态文件
    location / {
        root /var/www/admin-frontend/dist;
        try_files $uri $uri/ /index.html;
    }

    # API代理
    location /api/ {
        proxy_pass http://localhost:8001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # 上传文件
    location /uploads/ {
        alias /var/www/admin-backend/uploads/;
    }
}
```

#### 3. 后端部署
```bash
# 使用PM2
cd admin-backend
npm install --production
pm2 start ecosystem.config.js

# PM2配置文件
module.exports = {
  apps: [{
    name: 'admin-backend',
    script: 'src/app.js',
    instances: 2,
    exec_mode: 'cluster',
    env: {
      NODE_ENV: 'production',
      PORT: 8001
    }
  }]
}
```

#### 4. 数据库迁移
```bash

mysql -u username -p ai_xsread < admin_tables.sql
```
