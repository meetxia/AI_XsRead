## 📊 项目概览

### 基本信息
- **项目名称**: 文字之境后台管理系统
- **项目类型**: 前后端分离的管理后台
- **适用场景**: 小说阅读平台的内容管理和运营数据分析
- **开发周期**: 1天完成MVP版本
- **代码质量**: 企业级标准

### 技术栈总览

#### 前端技术栈
```
Vue 3.5.22 + Vite 7.1.12
├── UI框架: Element Plus 2.11.5
├── 状态管理: Pinia 3.0.3
├── 路由管理: Vue Router 4.6.3
├── HTTP客户端: Axios 1.12.2
├── 图表库: ECharts 6.0.0
├── 样式方案: Tailwind CSS 4.1.16
├── 日期处理: Day.js 1.11.18
└── 工具库: Lodash-es 4.17.21
```

#### 后端技术栈
```
Node.js + Express 5.0.1
├── 数据库: MySQL 5.7+ (mysql2 3.15.3)
├── 认证: JWT (jsonwebtoken 9.0.2)
├── 密码加密: bcryptjs 3.0.2
├── 文件上传: Multer 2.0.2
├── 安全防护: Helmet 8.0.2 + CORS 2.8.5
├── 速率限制: express-rate-limit 7.7.2
├── 日志: Morgan 1.10.0
└── 环境变量: dotenv 16.5.0
```

#### 数据库设计
```
MySQL 5.7+
├── 原有表: 12张 (用户、小说、章节等)
└── 新增表: 5张 (管理员、日志、轮播图、公告、统计)
```

## 🔐 安全特性

### 认证与授权
✅ **JWT Token认证**: 
- AccessToken: 2小时有效期
- RefreshToken: 7天有效期
- 自动刷新机制

✅ **密码安全**: 
- bcrypt加密 (10轮salt)
- 强密码策略
- 登录失败限制

✅ **权限控制**: 
- 基于角色的访问控制 (RBAC)
- 细粒度权限验证
- 操作权限日志

### 请求安全
✅ **速率限制**: 
- 全局: 15分钟/100次
- 登录: 15分钟/5次
- API: 15分钟/100次

✅ **CORS配置**: 
- 白名单域名
- 凭证支持
- 预检请求处理

✅ **Helmet安全头**: 
- XSS防护
- 点击劫持防护
- MIME类型嗅探防护

### 数据安全
✅ **SQL注入防护**: 参数化查询  
✅ **XSS防护**: 输入过滤  
✅ **CSRF防护**: Token验证  
✅ **文件上传安全**: 类型和大小限制
