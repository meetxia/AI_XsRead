## 🔧 技术实现

### 1. 状态管理
- 使用 Pinia 的 `useUserStore` 管理用户状态
- 使用 `useBookshelfStore` 获取书架数据
- 响应式数据自动更新UI

### 2. 组件化
- 复用 `AppHeader` 和 `BottomNav` 组件
- 模块化的功能区块设计
- 易于维护和扩展

### 3. 数据展示
- 使用计算属性动态获取用户信息
- 默认头像使用 DiceBear API
- 优雅的错误处理（头像加载失败自动使用默认头像）

### 4. 路由集成
- 平滑的页面跳转
- 未登录自动跳转到登录页
- 保持导航状态

## 💡 使用说明

### 用户信息显示
用户信息从 `useUserStore` 中获取，包括：
- `userInfo.avatar` - 头像URL
- `userInfo.nickname` - 昵称
- `userInfo.username` - 用户名
- `userInfo.bio` - 个人签名
- `userInfo.id` - 用户ID
- `userInfo.createdAt` - 注册时间

### 头像更换
点击头像右下角的相机图标，选择图片文件即可上传（目前使用本地预览）

### 功能跳转
- 点击书籍封面：跳转到书籍详情页
- 点击"查看全部"：跳转到完整书架页面
- 点击功能菜单项：执行对应操作（部分待实现）
- 点击设置项：进入对应设置页面（待实现）

### 退出登录
点击退出登录按钮，会弹出确认提示，确认后清除登录状态并跳转到登录页
