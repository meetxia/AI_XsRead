# 文字之境 UI 重构方案 Demo

> 文档日期：2026-05-17

本目录包含用户前端重构的 3 个视觉方向 demo，每个方案均为静态 HTML，使用 TailwindCSS CDN。
所有方案都遵循「移动端优先」原则，提供深色/浅色双主题切换。

## 目录结构

```
ui-redesign-demos/
├── index.html              # 总入口（三个方案切换）
├── option-a-minimal/       # 方案 A：极简书香
├── option-b-magazine/      # 方案 B：沉浸杂志
└── option-c-luxury/        # 方案 C：轻奢文艺
```

## 三个方案对比

| 维度 | A 极简书香 | B 沉浸杂志 | C 轻奢文艺 |
|------|------------|------------|------------|
| 视觉关键词 | 留白、宁静、Serif | 大图、密度、动感 | 暗调、金色、毛玻璃 |
| 主色 | 珊瑚红 #E85A4F | 玫粉渐变 #FF4D8D | 鎏金 #C9A961 |
| 底色（浅） | 暖米白 #FAF8F5 | 纯白 #FFFFFF | 奶油 #F5F1EA |
| 底色（深） | 墨蓝 #1A1B23 | 深灰 #0F0F12 | 深咖 #15110D |
| 字体策略 | Serif 优先 | Sans-serif 优先 | Serif + Sans 混排 |
| 卡片风格 | 极简圆角，无阴影 | 大图驱动，渐变叠层 | 毛玻璃，金色描边 |
| 适合人群 | 文艺向、安静阅读 | 年轻向、内容驱动 | 高端向、收藏感 |

## 共用页面

每个方案均包含以下 5 个核心页面：

1. **发现页（index.html）** — 首页推荐 + 分类入口
2. **详情页（detail.html）** — 小说封面、简介、章节列表
3. **阅读页（reading.html）** — 沉浸式阅读体验
4. **书架页（bookshelf.html）** — 我的收藏与阅读进度
5. **我的（profile.html）** — 个人中心

## 启动方式

```bash
# 任意静态服务器都行
cd ui-redesign-demos
python -m http.server 8080
# 浏览器访问 http://localhost:8080
```

或直接用 VS Code 的 Live Server 插件打开 `index.html`。

## 评估建议

打开总入口后，每个方案在手机和桌面都试一遍，重点关注：
- 移动端阅读体验是否舒适（字号、行距、点击区）
- 卡片信息密度是否合适
- 深浅色切换是否舒服
- 动效是否过度或欠缺
- 整体调性是否符合「女性向小说阅读平台」定位
