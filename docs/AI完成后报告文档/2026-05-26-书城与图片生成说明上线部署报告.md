## 9. 回滚预案

如需回滚，按以下顺序执行：

1. 停止用户后端和管理后端：

```bash
pm2 stop xsread-backend xsread-admin-backend
```

2. 回滚代码到部署前 commit：

```bash
cd /www/wwwroot/xs.momofx.cn
git reset --hard c4c742c1e780dfe52f6002e3775096dd08b24436
```

3. 恢复数据库：

```bash
mysql -u <生产数据库用户> -p ai_xsread < /www/wwwroot/xs.momofx.cn/backups/deploy_20260526_143220/ai_xsread_before_20260526_143220.sql
```

4. 恢复环境文件：

```bash
cp /www/wwwroot/xs.momofx.cn/backups/deploy_20260526_143220/backend.env.backup /www/wwwroot/xs.momofx.cn/backend/.env
cp /www/wwwroot/xs.momofx.cn/backups/deploy_20260526_143220/admin-backend.env.backup /www/wwwroot/xs.momofx.cn/admin-backend/.env
```

5. 重装生产依赖并重启：

```bash
cd /www/wwwroot/xs.momofx.cn/backend && npm ci --omit=dev
cd /www/wwwroot/xs.momofx.cn/admin-backend && npm ci --omit=dev
cd /www/wwwroot/xs.momofx.cn
pm2 start ecosystem.config.js
pm2 save
```

6. 回滚后重新执行：

```bash
curl -fsS http://127.0.0.1:8005/api/health
curl -fsS http://127.0.0.1:8001/api/health
```
