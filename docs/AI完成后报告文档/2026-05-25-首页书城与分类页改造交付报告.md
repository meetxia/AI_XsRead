# 首页书城与分类页改造交付报告

## 完成内容

本次按用户要求安排两个子代理并行修改用户端移动端信息架构，并由主代理完成集成验收。

1. 首页改造成移动端书城样式：
   - 移除首页移动端原品牌头部和顶部工具区。
   - PC 端保留原顶部导航，移动端通过响应式类隐藏。
   - 顶部改为粘性搜索框，搜索框下方增加可横滑分类标题导航。
   - 首页分类标题从跳转搜索页改为首页内切换，点击后刷新热门榜、横滑书籍区和瀑布流数据。
   - PC 端首页搜索/分类局部 header 取消整块深色底，改为透明静态布局以融入页面背景。
   - 移除“今日书城/挑一本能立刻读进去的故事”引导卡，首屏改为 1-8 名双列热门榜单。
   - 首页主体改为多组横滑书籍区，包括“女生热读”“新书抢先”“完结可追”。
   - 首页下方新增“猜你喜欢”双列书籍流，支持 `IntersectionObserver` 自动加载，并保留“加载更多”按钮兜底。
   - 首页原有“主编推荐、读者最爱、分榜预览、推荐书单”等大榜单不再保留在首页。

2. 发现页改为分类页：
   - `/recommend` 页面文案、SEO、路由标题从“发现”改为“分类”。
   - `/discover` 继续重定向到 `/recommend`，兼容旧入口。
   - 原发现页专题区块已移除。
   - 分类页展示顺序调整为：频道/分类入口 → 各种榜单 → 精选内容。
   - 从首页迁入榜单区：本期主编荐读、本周读者最爱、分榜速览、推荐书单。
   - 原精选列表保留，并放在榜单区下方。

3. 底部固定菜单调整：
   - 移动端底部菜单保持固定。
   - 菜单文案和顺序调整为：书城、分类、书架、我的。
   - “分类”入口继续使用 `/recommend`，并保留 `/discover` 高亮兼容。
