## 回滚预案

如需回滚：

1. 停止用户后端：

```bash
pm2 stop xsread-backend
```

2. 恢复数据库：

```bash
cd /www/wwwroot/xs.momofx.cn
mysql -u <生产用户> -p ai_xsread < backups/deploy_20260522_190803_txt_upload/ai_xsread_before_20260522_190803.sql
```

3. 恢复代码到部署前 commit：

```bash
cd /www/wwwroot/xs.momofx.cn
git reset --hard c4c742c1e780dfe52f6002e3775096dd08b24436
cd backend && npm ci --omit=dev
pm2 start ../ecosystem.config.js --only xsread-backend --update-env
pm2 save
```

注意：回滚数据库前必须保持服务停止，避免写入丢失。
