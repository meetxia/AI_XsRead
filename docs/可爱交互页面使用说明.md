# 💖 治愈小确幸 - 可爱交互网页使用说明

## 🎯 项目简介

这是一个专为小红书16-25岁年轻女性设计的治愈系交互网页，融合了2025年最新流行梗和可爱元素，提供丰富的互动体验。

## ✨ 核心特色

### 1. 2024-2025流行梗融入
- **公主请** 👑 - 优雅复古风格特效
- **yyds** ⭐ - 彩虹爆炸效果
- **绝绝子** 💕 - 爱心烟花特效
- **多巴胺配色** 🌈 - 活力星星雨
- **i人/e人选择** - 个性化主题切换

### 2. 丰富的交互功能
- **点击特效**：星星飞溅、爱心撒花、屏幕闪光
- **心情表情**：5种可爱表情选择，触发不同效果
- **音效系统**：可爱的点击音效（可开关）
- **震动反馈**：移动端触觉反馈

### 3. 隐藏彩蛋
- **连续点击3次**同一个按钮触发超级豪华特效
- **访问计数器**：记录访问次数，特殊次数有惊喜
  - 第1次访问：欢迎特效
  - 第10次访问：特别庆祝
  - 每5次访问：纪念特效
- **控制台彩蛋**：打开浏览器开发者工具查看

### 4. 个性化体验
- **i人主题**：柔和蓝紫色系，温柔治愈风
- **e人主题**：多巴胺配色，活力满满风
- 主题会改变页面背景渐变色

## 📱 使用方法

### 基础使用
1. **双击打开** `keai-interactive.html` 文件
2. 等待加载屏幕消失（约1秒）
3. 点击任意按钮开始互动

### 功能说明

#### 主要按钮
- **公主请** 👑：触发公主风特效 + 随机鼓励文案
- **yyds** ⭐：触发彩虹特效 + yyds文案
- **绝绝子** 💕：触发爱心爆炸 + 夸赞文案
- **多巴胺爆发** 🌈：触发星星雨 + 活力文案

#### 心情选择
点击任意表情图标，会有相应的文案和特效反馈：
- 😊 开心
- 🥰 爱心
- 🤩 激动
- 😽 可爱
- 😌 放松

#### i人/e人选择
- **i人按钮** 🏠：切换到安静温柔主题
- **e人按钮** 🎉：切换到活力多巴胺主题

#### 音效开关
- 点击右下角悬浮按钮 🔊/🔇
- 可以开启或关闭音效

### 彩蛋触发方法
1. **连续点击彩蛋**：3秒内连续点击同一按钮3次
2. **访问次数彩蛋**：多次访问页面（数据保存在本地）
3. **分享提示**：互动5次后会提示截图分享

## 🔧 自定义配置

### 修改配置（位于HTML文件中的CONFIG对象）

```javascript
const CONFIG = {
    soundEnabled: true,        // 音效开关
    vibrationEnabled: true,    // 震动反馈开关
    // ... 其他配置
};
```

### 修改配色
在CSS的`:root`变量中修改颜色：
```css
:root {
    --primary: #FFB6D9;      /* 主色 */
    --secondary: #FFC9E7;    /* 次色 */
    --accent: #FF85B3;       /* 强调色 */
    /* ... */
}
```

### 添加新文案
在`CONFIG.messages`对象中添加新的文案数组。
