# 文字之境（AI-XsRead）项目现状总览

文档日期：2026-05-17

## 1. 项目定位

文字之境（AI-XsRead）是一个面向女性用户的小说阅读平台，采用前后端分离架构，当前由四个核心应用构成：

1. 用户端前端：`ai-xsread-vue3`
2. 用户端后端：`backend`
3. 管理端前端：`admin-frontend`
4. 管理端后端：`admin-backend`

当前项目已经从“基础可用”进入“功能扩展与体验打磨”阶段。用户端主阅读链路、搜索、小说详情、书架、个人中心、关注作者、上传小说、账号安全等模块均已落地；管理端已具备内容、章节、用户、评论、看板与系统设置的基础能力。

## 3. 四端运行边界

### 用户端前端
- 开发端口：`3008`
- 代理：`/api`、`/uploads` -> `http://localhost:8005`

### 用户端后端
- 服务端口：`8005`
- 作用：为用户端提供小说、阅读、书架、评论、划线、段评、上传、作者、兴趣标签、成就等接口

### 管理端前端
- 开发端口：`3010`
- 代理：`/api` -> `http://localhost:8001`

### 管理端后端
- 服务端口：`8001`
- 作用：为管理后台提供登录、看板、小说、章节、用户、评论、上传等接口

1. 文档口径显著过期，历史说明中仍保留旧路径、旧端口与模板化描述。
