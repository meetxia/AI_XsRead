## 0. 文档说明

本能力矩阵覆盖以下范围：

- 用户前端（`ai-xsread-vue3/`）：女性向小说阅读 SPA。
- 用户后端（`backend/`）：阅读 / 用户 / 互动 / 会员 / SEO API。
- 管理前端（`admin-frontend/`）：内容运营与数据看板后台。
- 管理后端（`admin-backend/`）：仅服务管理前端的独立后台 API。

**不在本文档范围内**：单元测试覆盖率细节、迁移脚本逐条说明、Nginx 配置语法、构建产物体积。这些请走对应的"审查报告 / 数据库迁移说明 / 部署手册"。
