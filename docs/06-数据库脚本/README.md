# 数据库脚本目录说明

最后更新日期：2026-05-17

本目录存放 AI-XsRead 的数据库初始化、后台表扩展、开发种子数据、管理员账号修复和历史测试数据脚本。

请优先阅读：[2026-05-17-数据库脚本索引.md](./2026-05-17-数据库脚本索引.md)

## 快速入口

| 需求 | 优先使用 | 备注 |
|---|---|---|
| 本地空库建表 | `database_init.sql` | 会删除并重建核心表，谨慎执行 |
| 导入当前开发演示数据 | `seed_data_complete.sql` | 当前推荐的数据入口 |
| 补建后台管理表 | `admin_tables.sql` | 手工执行，包含后台表、视图、过程、触发器、索引 |
| 重置用户端 admin 密码 | `更新admin密码-admin123.sql` | 密码重置为 `admin123` |
| 创建本地测试账号 | `创建管理员账号.sql` | 会写入多个测试账号 |
| 查看旧版测试数据 | `test_data.sql`、`historical_test_data_backup.sql` | 备份参考，不作为默认入口 |

## 边界说明

- 当前目录没有自动迁移脚本。
- 当前 SQL 都需要人工确认后手工执行。
- 不要把历史测试数据当成当前初始化入口。
- 不要在需要保留数据的数据库上直接执行 `database_init.sql`。
