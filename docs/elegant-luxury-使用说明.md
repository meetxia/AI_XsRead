## ⚙️ 配置选项

文件内置了配置系统，可以轻松调整功能开关：

```javascript
const CONFIG = {
    theme: 'morandi',               // 主题（预留扩展）
    enableCustomCursor: true,       // 自定义光标（桌面端）
    enableSmoothScroll: true,       // 平滑滚动
    enableParallax: true,           // 视差效果
    animation: {
        speed: 1.2,                 // 动画速度（建议0.8-1.5）
        stagger: 0.1,               // 交错延迟
        ease: 'power3.out',         // 缓动函数
    },
    debug: false                    // 调试模式（开启后显示日志）
};
```

### 如何修改配置

1. 用文本编辑器打开 `elegant-luxury.html`
2. 找到 `<script>` 标签内的 `CONFIG` 对象（约552行）
3. 修改参数值
4. 保存文件，刷新浏览器

## 🔧 自定义指南

### 修改颜色
找到 `:root` CSS变量（约47行）：
```css
:root {
    --primary: #9DADBC;     /* 改成你想要的主色 */
    --secondary: #B09A8B;   /* 改成你想要的辅助色 */
}
```

### 修改文案
直接编辑HTML中的文字内容，保持优雅克制的风格。

### 添加内容区块
在 `</main>` 标签前添加新的 `<section>`：
```html
<section class="section section-sm">
    <div class="container-narrow">
        <h2>新区块标题</h2>
        <p>新区块内容</p>
    </div>
</section>
```

### 调整动画速度
修改 `CONFIG.animation.speed`（越大越慢）：
```javascript
animation: {
    speed: 1.5,  // 默认1.2，建议范围0.8-2.0
}
```

## 🐛 常见问题

### Q1：动画不流畅？
**A**：检查以下几点：
- 确保浏览器支持硬件加速
- 关闭过多的浏览器标签页
- 更新浏览器到最新版本

### Q2：光标看不见？
**A**：自定义光标仅在**桌面端**（屏幕宽度>1024px）显示。
移动端会自动隐藏。

### Q3：字体加载慢？
**A**：Google Fonts从国内访问可能较慢，可以：
- 等待字体加载完成
- 或替换为国内CDN字体

### Q4：滚动动画不触发？
**A**：打开调试模式查看日志：
```javascript
const CONFIG = {
    debug: true  // 改为true
};
```

## 💡 设计心得

作为一名设计大师和用户心理学专家，我在创作这个作品时遵循了以下原则：

### 1. 理解目标用户
25-40岁成熟女性用户的审美特点：
- 已过了追求视觉刺激的阶段
- 更注重质感和细节
- 喜欢留白和呼吸感
- 反感花哨和过度装饰
- 追求内在的优雅和从容

### 2. 克制的设计语言
- 每一个颜色都经过精心挑选
- 每一个间距都符合黄金比例
- 每一个动画都有存在的理由
- 每一句文案都经过反复推敲

### 3. 细节的力量
- 按钮的圆角是4px而不是8px（更克制）
- 卡片悬停时只上移4px而不是8px（更微妙）
- 动画时长是1.2秒而不是0.5秒（更优雅）
- 分隔线宽度是60px而不是100%（更有节奏）

### 4. 情感化设计
通过文案和视觉的配合，让用户感受到：
- "这个设计懂我"
- "终于有人理解我的审美"
- "我想把这个保存下来慢慢欣赏"
