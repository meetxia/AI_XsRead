## ❓ 常见问题

### Q1：HTML文件打不开？
**A**：右键 → 打开方式 → 选择浏览器（Chrome/Edge/Firefox）

### Q2：使用CDN需要联网吗？
**A**：是的，首次需要联网加载库，之后浏览器会缓存

### Q3：如何完全离线使用？
**A**：选择"纯原生"方案，或下载CDN文件到本地

### Q4：Python脚本报错缺少库？
**A**：运行 `pip install -r requirements.txt` 安装依赖

### Q5：可以修改颜色和文字吗？
**A**：可以！找到代码顶部的CONFIG配置区域修改即可

### Q6：效果不够震撼怎么办？
**A**：使用HTML5 + Three.js组合，3D效果最震撼

### Q7：想要加照片怎么办？
**A**：告诉AI"我想加入我们的照片"，AI会帮你实现
