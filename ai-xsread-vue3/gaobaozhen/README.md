# 文字之境 - 女性向小说阅读网站

> 故事入境，杂念自消

## 🌸 项目简介

文字之境是一个专为女性读者打造的高保真小说阅读网站。采用现代化设计理念，以温柔而有力的视觉语言，为读者营造一个纯粹的阅读空间。

## ✨ 核心特点

### 设计理念
- **卡片为门** - 一眼知书。标题、字数、心意，皆在方寸
- **瀑布为序** - 故事如流水，自上而下，层叠不息
- **触手可及** - 移动优先，指尖阅尽千言
- **女性叙事** - 温柔有力的视觉语言

### 三大运行律
1. **聚合律** - 首页如展厅，每张卡片都是邀请函
2. **沉浸律** - 阅读时，世界只剩文字与呼吸
3. **归属律** - 书架是港湾，每本书都等待重逢

### 主题切换方式

1. **桌面端**：点击顶部导航栏的"主题"按钮，在下拉菜单中选择
2. **移动端**：暂时使用默认主题，未来将支持移动端切换
3. **暗色模式**：点击导航栏右侧的月亮图标切换明暗模式
4. **自动保存**：所有主题设置自动保存到本地，下次访问时恢复

## 🚀 使用方法

### 方式1：直接打开
```bash
# 直接用浏览器打开
vue_xs/frontend/public/xs-ui-/index.html
```

### 方式2：本地服务器（推荐）
```bash
cd vue_xs/frontend/public/xs-ui-
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 方式3：使用 Live Server（VS Code）
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🔧 开发指南

### 添加新主题
1. 在 `theme.js` 的 `themeConfig` 中添加新主题配置
2. 在导航栏下拉菜单中添加对应按钮
3. 测试明亮和暗色两种模式

### 修改现有主题
1. 打开 `theme.js`
2. 找到对应主题的配置
3. 修改颜色值
4. 保存后刷新页面即可看到效果

### 调试主题系统
```javascript
// 在浏览器控制台中
console.log(window.themeManager.getCurrentTheme());
// 查看当前主题配置

window.themeManager.setTheme('vintage');
// 切换到文物古韵主题

window.themeManager.toggleMode();
// 切换明暗模式
```
